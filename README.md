## Running Locally

Simply open `index.html` in any modern browser (Chrome, Edge, Firefox).

## Deploying to Appwrite Sites

This vanilla HTML/CSS/JS site is fully ready to be deployed natively on **Appwrite Sites**. No build process is required.

**Via GitHub Integration (Recommended)**
1. Go to your Appwrite Console > **Sites**.
2. Click **Create site** and select **Connect a repository**.
3. Select your GitHub repository (`balirwaalvin/chrysm`).
4. **Configuration settings:**
   - **Build command:** `npm run build` (This is handled gracefully via the dummy script in `package.json`)
   - **Output directory:** `.` (Keep as the root directory since `index.html` is at the root level).
5. Deploy! Every branch push will automatically deploy preview or production links.
